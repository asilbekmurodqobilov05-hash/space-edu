#### 1. 🌌 3D Солнечная система (Real-Time)
- **Технологии:** React Three Fiber + Three.js + GLSL шейдеры
- **Требования:**
  - Все 8 планет с реальными текстурами (NASA texture maps), правильными наклонами осей и кольцами (Сатурн, Уран)
  - Реальные орбитальные параметры: большая полуось, эксцентриситет, наклонение, период обращения — из NASA JPL Horizons
  - Расчёт позиций планет на текущую дату через Кеплерову модель (mean anomaly → eccentric anomaly → true anomaly → heliocentric coordinates)
  - Карликовые планеты: Плутон, Церера, Эрида, Хаумеа, Макемаке
  - Пояс астероидов (процедурная генерация ~5000 частиц)
  - Луны основных планет (минимум: Луна, Фобос, Деймос, Ио, Европа, Ганимед, Каллисто, Титан, Энцелад, Тритон)
  - Управление камерой: свободный полёт, привязка к планете, следование за объектом
  - Временная шкала: ускорение/замедление времени, перемотка на произвольную дату
  - UI-панель с данными: расстояние от Солнца, температура, масса, период обращения, гравитация, атмосфера
  - Skybox из реальных звёздных данных (Tycho-2 / Hipparcos каталог)

### Правила разработки:

1. **Изучи прежде чем писать.** Перед любым изменением — прочти и проанализируй существующий код. Пойми архитектуру, паттерны, зависимости. Никогда не пиши вслепую.

2. **Научная точность.** Все физические величины, орбитальные параметры, исторические факты — должны быть верифицированы. Используй данные NASA JPL, IAU, ESA. Неточности недопустимы.

3. **Перформанс 3D.** 
   - Бюджет: 60fps на mid-range GPU
   - LOD (Level of Detail) для планет — высокая детализация вблизи, low-poly на расстоянии
   - Instanced meshes для астероидного пояса и звёзд
   - Frustum culling, texture compression, lazy loading текстур
   - Web Workers для орбитальных расчётов
   - Offscreen Canvas для тяжёлых рендеров

## 📎 API И ИСТОЧНИКИ ДАННЫХ

- **NASA Open APIs:** https://api.nasa.gov (APOD, Mars Rover, NEO, Exoplanet)
- **NASA JPL Horizons:** эфемериды для точных позиций
- **ISS Location:** http://api.open-notify.org/iss-now.json
- **SpaceX API:** https://api.spacexdata.com/v5
- **Launch Library 2:** https://ll.thespacedevs.com/2.2.0
- **NASA Texture Maps:** https://svs.gsfc.nasa.gov/
- **Stellarium Web:** для данных о созвездиях и Deep Sky
